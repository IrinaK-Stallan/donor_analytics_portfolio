# 📊 Анализ данных о донорстве крови

**Цель проекта** — выявить ключевые тренды активности доноров, популярность бонусов, эффективность планирования и факторы удержания.

## 🛠 Инструменты

- **SQL (PostgreSQL)**: оконные функции, CTE, агрегации
- **Excel**: сводные таблицы, графики, условное форматирование

## 📁 Структура проекта

**1_Регионы** — Топ регионов по числу доноров  
*Вывод:* Москва + СПБ + Казань = 55% доноров

**2_Динамика** — Помесячная динамика донаций  
*Вывод:* В 2023 спад после июня

**3_Активные доноры** — Рейтинг доноров по активности  
*Вывод:* Донор-4 делает 19,6 донаций в год (ядро)

**4_Бонусы** — Популярность бонусов  
*Вывод:* Пицца и подписки популярны, но возможна директивность

**5_Соцсети** — Привязанные аккаунты и активность  
*Вывод:* VK — лидер, но активнее доноры с Google-аккаунтом

**6_Retention** — Однократные vs многократные доноры  
*Вывод:* Многократные дают 90% всех донаций

**7_Планы_vs_факты** — Эффективность планирования  
*Вывод:* Только 3.6% донаций запланировано, но доля растёт
